# react-frontend-test
React Frontend Test

Setup:

1.) Clone Repo  
2.) npm install  
3.) npm start  

Note:  

1.) The frontend side is traight forward, once it runs, it will already show a list of users.  
2.) Make sure to run the Test-Api server first.  
3.) Clicking a user will show a straight forward success alert and it will be added to the Test-Api database.
